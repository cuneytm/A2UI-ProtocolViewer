## ğŸ› ï¸ Architecture

1.  **User**: Sends a natural language prompt (e.g., "Analyze ETH").
2.  **Server (Express)**: Manages SSE (Server-Sent Events) connections.
3.  **Coordinator (Gemini)**: Analyzes the prompt and delegates work to specialized agents.
4.  **Specialized Agents**: 
    *   **Market Data Agent**: Searches for price data and generates A2UI Charts.
    *   **News Agent**: Searches for current events and generates A2UI Article Cards.
5.  **Renderer**: A client-side JavaScript engine that interprets the A2UI protocol and paints the DOM.

## ğŸ Getting Started

### 1. Prerequisites
- Node.js 18+
- [Google AI Studio API Key](https://aistudio.google.com/app/apikey)

### 2. Installation
```bash
git clone https://github.com/cuneytm/A2UI-ProtocolViewer.git
cd A2UI-ProtocolViewer
npm install
```

### 3. Environment Setup
Create a `.env` file or export your key:
```bash
export GEMINI_API_KEY="your_api_key_here"
```

### 4. Run the App
```bash
npm start
```
Navigate to `http://localhost:3000` to see the protocol in action!

## ğŸ“¡ A2UI Protocol in Action

A2UI (Agent-to-UI) allows agents to control their own presentation. Instead of text, agents emit JSONL:

```json
{"surfaceUpdate": {"components": [{"id": "chart", "component": {"Chart": {"type": "line", "data": {...}}}}]}}
{"beginRendering": {"root": "chart"}}
```

The viewer renders these messages instantly, creating a fluid, agent-decided experience.
